# Snagit Clone

A powerful screen capture and editing tool built with Python and PySide6.

## Features
- **Capture**: Full screen, region, window capture.
- **Edit**: Annotate with arrows, shapes, text, blur.
- **Share**: Copy to clipboard, save to file.

## Setup
1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Run operation:
   ```bash
   python src/main.py
   ```

## Project Structure
- \`src/\`: Source code (main application logic, widgets, utils)
- \`dist/\`: Compiled executable (if built)
- \`build/\`: Build artifacts
- \`run.bat\`: Helper script to run the app
- \`requirements.txt\`: Python dependencies
